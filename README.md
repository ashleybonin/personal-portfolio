# Personal Portfolio Website

A responsive portfolio website built using HTML, CSS, and JavaScript to showcase my technical projects, data analysis work, and front-end development skills.

## Overview

This project highlights my ability to design and develop a clean, user-focused interface while implementing interactive functionality. The site includes structured project cards, working navigation, and smooth scrolling for an enhanced user experience.

## Features

- Responsive layout design
- Interactive navigation menu
- Smooth scrolling functionality
- Project section with linked repositories
- Clean and organized front-end code structure

## Technologies Used

- HTML
- CSS
- JavaScript
